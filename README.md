# Xilinx FPGA SRIO 接口 Verilog 源码程序

本仓库提供了一个基于 Xilinx FPGA 的 SRIO 接口 Verilog 源码程序。该源码程序的顶层接口封装为 FIFO，使用简单方便，并且已经在实际项目中得到了应用。本源码支持 SRIO 的 NWRITE、NWRITE_R、SWRITE、MAINTENCE、DOORBELL 等事务。
